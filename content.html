Abstract

A Dynamic Website is the most widely used type of website used by businesses and organizations to interact with customers. A dynamic website uses server-side processing and technologies such as PHP, Java, or .NET to build web pages that new customized as per the visitor requests.

A CMS (Content Management System) can be used to create a dynamic website. Amazon Web Services (AWS) offers cloud website solutions that provide businesses, non-profits, and governmental organizations with a flexible, highly scalable, and low-cost way to deliver their websites and web applications.

Technologies used:

  • Wordpress
  • Amazon Web Services (Elastic Cloud Compute)
